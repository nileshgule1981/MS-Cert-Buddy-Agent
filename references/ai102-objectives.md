# AI-102: Designing and Implementing a Microsoft Azure AI Solution

**Skills Measured (as of April 2025)**

## Plan and manage an Azure AI solution (25-30%)

### Plan and manage Azure AI services

- Select the appropriate Azure AI service for a solution
- Plan and configure security for an Azure AI service
- Plan for an Azure AI service that meets cost requirements
- Plan for an Azure AI service that meets compliance requirements
- Plan for an Azure AI service that meets responsible AI principles
- Manage Azure AI service costs
- Monitor an Azure AI service
- Deploy Azure AI services by using containers

### Manage Azure AI services security

- Manage authentication for an Azure AI service
- Manage access control for an Azure AI service
- Secure Azure AI services by using Azure Key Vault
- Manage Azure AI service keys
- Implement network security for Azure AI services
- Manage private endpoints for Azure AI services
